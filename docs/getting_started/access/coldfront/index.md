# User and Allocation Management

[Coldfront](https://coldfront.otago.ac.nz) is a web interface that enables project owners (and delegates) to manage adding/removing user access to their group resources such as project storage.

- New users will need to apply for an account to be created to access the cluster
- Exisiting users can then either:
    - be added to an existing group with a project allocation by the project owner/delegate
    - or have their group leader apply for a project allocation

Alterations to projects are managed from the Coldfront interface by project owners/delegates who can submit requests to change current storage quotas, apply for new allocations (e.g. WEKA access), or add/remove group members.

This is currently in developement and until implemented user management is done through the [user sign up form](https://forms.office.com/pages/responsepage.aspx?id=xe8lAv54KEmxV57ySAnpuklJ-oAsUjlEj9BHqdUHZWtUN0xCOVhOSEdUS1ZNNllUNENWVEpBSUtEUS4u&route=shorturl) (for new users) or through emailing {{ support_email }}.
