# Database hosting


eResearch Solutions can host your **research** database, for individual researchers or small groups.

## PostgreSQL

These can be directly interfaced with e.g. via Notebook/python/psql/etc, or via our [pgAdmin web UI](https://pgadmin.rtis-k8s-delta.uod.otago.ac.nz).


## MariaDB (MySQL)

These can be directly interfaced with e.g. via Notebook/python/mysql/mariadb/etc or any mysql client of your choice.
